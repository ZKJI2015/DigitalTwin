## 本机与局域网访问

- 重新启动 `0.0.0.0:8088` 静态服务器，并新增 `start-server.bat` / `start-server.ps1` 启动脚本，方便用户自行启动。
- 添加 Windows 防火墙规则需用户手动执行管理员命令：`netsh advfirewall firewall add rule name="DigitalTwin HTTP 8088" dir=in action=allow protocol=TCP localport=8088`。
- 若系统开启 Clash 等代理，需将 `192.168.20.73`/`127.0.0.1` 加入代理 bypass，或临时关闭系统代理。

## 升级为 VR 全景（WebXR）

- 应用户要求「全景换成VR全景」，将普通 360° 全景升级为支持 VR 头显的 WebXR 沉浸式全景。
- Three.js 由 r128（全局脚本）升级到 r160（ES 模块 + importmap），启用 `renderer.xr.enabled` 与 `VRButton`，渲染循环改用 `renderer.setAnimationLoop()`。
- 热点由 DOM 元素改为 3D 网格（发光球 + 双脉冲环），桌面用鼠标射线点击 → DOM 弹窗，VR 用凝视停留 1.2s → 3D 信息面板（CanvasTexture），另支持手柄射线选择。
- 控制改造：OrbitControls 改为原地旋转（target=原点），滚轮改为 FOV 缩放（正确的全景缩放方式）；进入 VR 时禁用 OrbitControls 并隐藏桌面 HUD。
- 重要限制：WebXR 仅在安全上下文可用——`http://localhost` 可用，局域网 IP（http://192.168.x.x）需 HTTPS 才能在手机上进入 VR；桌面本机经 localhost 即可进入 VR。
