# DigitalTwin — 本地部署与局域网访问说明

快速将项目部署到本机并在局域网中可访问。

启动步骤：

1. 安装 Node.js（推荐 v14+）。
2. 在项目根目录打开终端，运行：

```
npm install --no-audit --no-fund
npm start
```

默认会监听 `0.0.0.0:8080`，在局域网内其他设备可通过 `http://<主机局域网IP>:8080/` 访问。

可选端口/绑定地址：

```
HOST=0.0.0.0 PORT=3000 npm start
```

防火墙注意：

- Windows 防火墙可能会阻止端口，请允许 `node.exe` 或开放相应端口。
- 如果使用路由器隔离或 VPN，确保设备在同一 LAN 子网下。

测试：

- 在本机浏览器打开 `http://localhost:8080/`。
- 在另一台局域网设备上打开 `http://<主机IP>:8080/`。
